# Эксплуатация беспилотных авиационных систем
## Назначение
Система предназначена для сельскохозяйственной сферы. Она подразумевает под собой возможность собирать и транспортировать промышленные объекты с помощью БПЛА, а также в нее входит обслуживание линий электропередач с помощью БПЛА, что включает в себя посадку на провод, выявление дефектов и его отчистку за счет скольжения на дополнительно спроектированном устройстве.

#### Ключевые инженерные решения 
Ключевыми инженерными решениями по отношению к модулю А являются: <br>
•	Устройство подходит для всех трех заданных грузов. <br>
•	Это решение обеспечивает надежность сбора, транспортировки и посадки грузов, а также универсальность, поскольку ширину разведения крюков регулирует оператор. <br>
